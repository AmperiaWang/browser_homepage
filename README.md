# 适用于各式设备的浏览器主页

由于手机上的浏览器主页不够简洁（收藏夹只有极小的一栏，而且下面的那些资讯我也不想看），所以我就写了这个浏览器页面。

这个仓库与[我位于Gitee上的仓库](https://gitee.com/ericwang017/browserhome)是同步的。遵循的协议也相同（均为MIT协议）。如果有感兴趣的也可以fork出去改。所有的代码就那一个HTML文件（库是从国内的CDN调的），顶多再加一个favicon，在里面我也做了一些注释，因此从接触这个项目到上手可能几个小时就能搞定（毕竟我是想做的简单一点，写这个项目也没用多少时间）。

说是适用于各式设备，其实就是一个响应式的页面。绝大部分支持HTML5的浏览器都能成功运行（当然如果你碰巧点背的话，可以把浏览器型号通过issues发给我，我有空的话去尽量做个适配）。如果你的设备真的太老了，也别强求，适合的才是最好的。所有的用户数据都通过localStorage存储在本地（没有经费，也没有精力，更没有必要去搞服务器），因此您可以不必担心隐私问题，放心大胆地使用。
